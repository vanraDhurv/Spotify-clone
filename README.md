# 🎵 Spotify Clone (HTML & CSS)

A responsive front-end clone of the Spotify Web Player built using pure HTML and CSS. This project focuses on recreating the user interface and layout of Spotify, including sidebar navigation, music player controls, and content sections.

## 🚀 Features
- 🎧 Spotify-inspired UI design
- 📂 Sidebar with navigation (Home, Search, Library)
- 📀 Music player section with controls
- 📊 Multiple content sections (Recently Played, Trending, Featured)
- 📱 Responsive design for different screen sizes
- 🎨 Modern UI using Flexbox and CSS styling

## 🛠️ Tech Stack
- HTML5
- CSS3 (Flexbox, Responsive Design)
- Font Awesome (icons)
- Google Fonts
